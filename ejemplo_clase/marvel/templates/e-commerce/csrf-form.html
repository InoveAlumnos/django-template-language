Prueba de etiqueta "csrf_token"

Los formularios de tipo POST necesitan enviarse con esta etiqueta, que envía un token de seguridad para evitar que bots u otros usuarios envíen datos al endpoint de "action" a nuestros formularios. Más información acerca de CSRF

Dado el siguiente formulario CON ETIQUETA "csrf_token":

{% csrf_token %}




Ahora el formulario SIN "csrf_token":






Información del request:

    {% for key, value in request.POST.items%}
  • {{key}} : {{value}}
    • {% endfor %}
