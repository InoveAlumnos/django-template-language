Prueba de bucle FOR

El bucle for en los templates se utilizan en mayor medida para generar listas o tablas que tienen un contenido dinámico

1. Probamos en una variable personalizada en el contexto:

Dada la siguiente lista:

{{TEST_LIST}}

Iteramos:

En medio de un parrafo, podríamos incluir un mensaje como: " {% for item in TEST_LIST %} {{item}} {% endfor %} " y luego cerrar.

2. Probamos en otra variable personalizada en el contexto:

Dado el siguiente diccionario:

{{TEST_DIC}}

Iteramos:

    {% for k, v in TEST_DIC.items %}
  • {{k}} : {{v}}
    • {% endfor %}

3. Utilizamos el FOR para obtener los valores de la variable "request.headers"

    {% for k,v in request.headers.items %}
  • {{k}} : {{v}}
    • {% endfor %}
