Variables personalizadas en context

La función get_context_data() puede ser sobrecargada para agregar la información y los procesos que necesitemos

Variable personalizada prueba_lista

Para acceder a las listas debemos utilizar los índices:

  • prueba_lista.0 : {{prueba_lista.0}}
  • prueba_lista.1 : {{prueba_lista.1}}
  • prueba_lista.2 : {{prueba_lista.2}}

Variable personalizada prueba_diccionario

Para acceder a las listas debemos utilizar los índices:

  • prueba_diccionario.0 : {{prueba_diccionario.saludo}}
  • prueba_diccionario.1 : {{prueba_diccionario.usuario}}
