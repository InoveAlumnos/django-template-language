Prueba de sentencia IF

La sentencia IF, admite también las varientes ELIF y ELSE

1. Probamos la sentencia IF/ELSE:

Dada la siguiente lista:

{{TEST_LIST}}

Consultamos:

{% if 'Inove ' in TEST_LIST %}

Existe Inove en la lista :D

{% else %}

No existe Inove en la lista :C

{% endif %}

2. Probamos la sentencia IF/ELIF/ELSE con la variable "user":

La variable "user" va a tomar el valor del usuario que ha iniciado sesión en el sitio.

Consultamos:

{% if user.is_superuser %}

Bienvenido superusuario {{user.username}}

{% comment %} AnonymousUser {% endcomment %} {% elif user.username != '' %}

Bienvenido {{user.username}}

{% else %}

Usuario no registrado --> AnonymousUser, por favor, inicie sesión para poder navegar.

{% endif %}
